## Deployment

The Neural Nexus web application is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

### Manual Deployment

You can manually trigger a deployment at any time:

1. Go to the [Actions tab](https://github.com/AndreasMoerch/consciousness/actions)
2. Select the "Deploy Neural Nexus to Pages" workflow
3. Click "Run workflow" button
4. Select the `main` branch (or another branch if needed)
5. Click "Run workflow" to start the deployment

The deployment typically takes 1-2 minutes to complete. Once finished, the site will be available at:
https://andreasmoerch.github.io/consciousness/
